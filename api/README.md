## CRUD API Rest Spring Boot 3

CRUD de uma API Rest usando o Spring Boot 3, com armazenamento das informações em um banco de dados MYSQL e com a biblioteca Flyway para controle de Migration.

---

### Tecnologias utilizadas

- Spring Boot 3
- Java 17
- Lombok
- MySQL/ Flyway
- JPA/Hibernate
- Maven
- Postman
